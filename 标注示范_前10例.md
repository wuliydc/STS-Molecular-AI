## 常见判断规则速查

| 情况 | 判断 |
|------|------|
| 阳性细胞比例 ≥15% | 检测结果 = 阳性 |
| 阳性细胞比例 <15% | 检测结果 = 阴性 |
| "未显示...基因易位" | 检测结果 = 阴性 |
| "显示XX基因易位/融合" | 检测结果 = 阳性 |
| "形态符合/考虑/倾向" | 诊断是否明确 = 倾向性诊断 |
| "建议补充检测" | 诊断是否明确 = 待明确 |
| 大体描述为外院切片编号 | 检测方法 = 会诊 |
| "肿瘤常见基因易位检测" | 检测方法 = RNA-NGS |
| "EGFR/KRAS/BRAF..." | 检测方法 = DNA-NGS |
